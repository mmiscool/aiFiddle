Getting Started with aiFiddle

aiFiddle lets you interact with various LLMs directly in the browser. To get started, youâ€™ll need API keys for one or more of the supported providers.

Supported Providers

OpenAI

1. Visit OpenAI API Keys.

2. Log in and create a new API key.

3. Copy the key and paste it into aiFiddle when prompted or use the settings panel.

Note: GPT-4 usage may require a paid plan.

Google Gemini

1. Navigate to Google AI Studio.

2. Sign in with your Google account and generate a key.

3. Paste the API key into aiFiddle's settings or prompt box.

Groq

1. Go to Groq Console (login required).

2. Create an API key.

3. Paste it into the aiFiddle LLM config menu.

xAI (Grok)

1. Log in to x.ai Platform (or X Developer portal).

2. Follow the instructions to obtain an API key for Grok access.

3. Use the key inside aiFiddle's settings under the xAI provider section.

Note: Access may be limited depending on account verification.

Where to Enter API Keys

Look for the gear/settings icon in the upper right corner of aiFiddle. You can paste API keys there for each provider.
